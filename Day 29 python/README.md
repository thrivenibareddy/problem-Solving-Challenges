# Day 29 – Student Marks Analyzer (Python Dictionary)

## Overview

This project is part of my **Python Coding Challenge Series**.
In Day 29, I built a **Student Marks Analyzer** using Python dictionaries to manage and analyze student academic data.

The program stores student marks, calculates performance statistics, identifies the top scorer, and detects failed students automatically.

---

## Objectives

* Practice Python **Dictionary** concepts
* Work with **nested dictionaries**
* Perform data analysis using Python
* Improve problem-solving skills

---

##  Concepts Used

* Dictionary & Nested Dictionary
* Loops (`for`)
* Built-in functions (`sum()`, `len()`, `max()`)
* Conditional statements
* List operations

---

## Problem Statement

Given student marks in different subjects:

* Calculate **Total Marks**
* Calculate **Average Marks**
* Identify **Top Scorer**
* Find **Failed Students** (marks < 40 in any subject)

---

##  Sample Data

```python
students = {
    "Asha": {"Math": 78, "Science": 85, "English": 67},
    "Ravi": {"Math": 35, "Science": 40, "English": 50},
    "John": {"Math": 90, "Science": 92, "English": 88}
}
```

---

##  Features

✅ Calculates total marks for each student
✅ Computes average marks
✅ Finds highest scoring student
✅ Detects failed students automatically

---

##  Output Example

```
Asha → Total: 230, Average: 76.67
Ravi → Total: 125, Average: 41.67
John → Total: 270, Average: 90.00

Top Scorer: John
Failed Students: ['Ravi']
```
