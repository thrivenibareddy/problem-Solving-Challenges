# Day 27 - Ride Booking Data Analysis (Python Tuples)

##  Challenge
Analyze ride booking trip data using Python tuples.

Each trip contains:
(Trip_ID, City, Distance_km, Fare_amount)


##  Dataset

``python
trips = (
    (101, "Bangalore", 12.5, 250),
    (102, "Hyderabad", 8.0, 180),
    (103, "Bangalore", 15.2, 300),
    (104, "Chennai", 5.5, 120),
    (105, "Bangalore", 7.8, 200)
)

**## Tasks Completed**
Total number of trips
Count Bangalore trips
Total revenue calculation
High distance trips (>10 km)
Highest fare trip
Average fare for Bangalore trips

**Concepts Used**
Tuples
Looping
Conditional Filtering
Aggregation
Lambda Functions

Learning Outcome

This project demonstrates basic data analysis logic using core Python concepts similar to real-world ride-booking analytics.
