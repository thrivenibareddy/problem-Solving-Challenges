# IT Support Executive Dashboard 

##  Overview
The **IT Support Executive Dashboard** is an Excel-based analytics solution designed to monitor, analyze, and improve help desk operations. It provides key insights into ticket volume, resolution efficiency, analyst performance, and recurring issues.

This dashboard enables IT managers and stakeholders to make **data-driven decisions** and optimize support processes.



##  Dataset Description
The dataset includes the following fields:

- **ticket_id**: Unique identifier for each ticket  
- **submission_date**: Date and time the ticket was created  
- **resolution_date**: Date and time the ticket was resolved  
- **category**: Type of issue (Access, Billing, Hardware, Network, etc.)  
- **assigned_analyst**: IT staff assigned to the ticket  
- **description**: Short problem description  
- **priority**: Ticket urgency level (Low, Medium, High, Critical)  



##  Data Preparation
Additional calculated fields were created in Excel:

- **Resolution Time (Hours)**  
- **Day of Week**  
- **Month**  
- **Hour of Day**  

These fields support time-based and performance analysis.



##  Dashboard Features

### Visualizations
- **Ticket Volume Trend (Hourly)**  
- **Resolution Time by Category & Priority**  
- **Analyst Performance (Avg Resolution Time vs Ticket Count)**  
- **Ticket Distribution by Day of Week**  


### Interactive Filters (Slicers)
- Category  
- Priority  
- Month  

These allow dynamic exploration of the data.


## Key Insights
- Identifies peak support hours and busiest days  
- Highlights categories with the highest workload  
- Compares resolution efficiency across analysts  
- Detects potential bottlenecks in ticket handling  


##  Recommendations
- Automate repetitive issues  
- Provide targeted training for slower-performing analysts  
- Allocate resources during peak hours  
- Improve processes for high-resolution-time categories
