# 📊 MGNREGS Performance Analysis Dashboard

## 📌 Overview

This project presents an interactive **Power BI dashboard** analyzing the performance of the **MGNREGS (Mahatma Gandhi National Rural Employment Guarantee Scheme)** across different states and years.

The dashboard provides insights into employment generation, expenditure efficiency, and work completion trends to support data-driven decision-making.


## 🎯 Objectives

* Analyze **employment generation (Person-Days)**
* Compare **household demand vs provided work**
* Evaluate **expenditure efficiency**
* Identify **top and underperforming states**
* Provide actionable **insights & recommendations**



## 📂 Dataset

The dataset includes:

* Year-wise data (2017–2021)
* State-level metrics
* Household demand & provided
* Expenditure (in crores)
* Person-days generated



## 🛠️ Tools & Technologies

* **Power BI** – Data visualization & dashboard creation
* **DAX (Data Analysis Expressions)** – Calculations & measures
* **Excel / CSV** – Data source



## 📊 Dashboard Features

### 🔹 KPI Cards

* **44M** – Total Household Demand
* **38M** – Total Household Provided
* **122K Cr** – Total Expenditure
* **87.79%** – Work Completion Rate



### 📈 Visualizations

#### 1. Expenditure Trend

* Line chart showing **Expenditure per Person Day over years**
* Identifies growth patterns and fluctuations

#### 2. Geographic Analysis

* Map visualization showing **state-wise expenditure distribution**

#### 3. State Comparison

* Bar chart comparing **Work Completion Rate by State**
* Highlights high vs low performing states

#### 4. Demand vs Provided

* Pie chart showing **distribution across years**

#### 5. Tabular Insights

* Table displaying **Person-Days generated by state**



## 📌 Key Insights

* 📈 **Expenditure per person day is increasing**, especially after 2019
* 🏆 **Uttar Pradesh leads** in person-days generated
* ⚖️ **Work completion rate remains consistently high (~85–90%)** across states
* ⚠️ Some states show **gap between demand and provided work**



## 📊 DAX Measures Used

```DAX
Work Completion Rate = 
DIVIDE(SUM('Table'[Households_Provided]), SUM('Table'[Households_Demanded]))

Expenditure per Person Day = 
DIVIDE(SUM('Table'[Total_Expenditure]), SUM('Table'[Person_Days_Generated]))

Total Person Days = 
SUM('Table'[Person_Days_Generated])
```


## 📌 Insights & Recommendations

* Improve **resource allocation** in underperforming states
* Focus on **reducing demand-supply gaps**
* Monitor **high expenditure vs low output regions**
* Enhance **policy planning using data-driven insights**
