# 🏥 Medical Insurance Cost Analysis (SQL Project)

## 📌 Project Overview

This project analyzes medical insurance data using SQL to extract meaningful insights about insurance costs based on various factors such as age, gender, BMI, smoking habits, and region.

The goal is to demonstrate strong SQL skills including filtering, aggregation, grouping, and conditional analysis.


## 🗂️ Dataset Description

The dataset contains the following fields:

| Column Name    | Description                     |
| -------------- | ------------------------------- |
| policy_id      | Unique ID for each policyholder |
| age            | Age of the individual           |
| gender         | Male / Female                   |
| bmi            | Body Mass Index                 |
| children       | Number of dependents            |
| smoker         | Yes / No                        |
| region         | Geographic region               |
| insurance_cost | Medical insurance cost          |

## 🔍 Key SQL Tasks Performed

1. Count total number of records
2. Filter smokers above age 40
3. Count policyholders by gender
4. Compare average insurance cost (smokers vs non-smokers)
5. Identify region with highest average cost
6. Retrieve top 5 highest insurance costs
7. Group and analyze cost by age categories
8. Analyze BMI impact on smokers vs non-smokers
9. Generate insights using GROUP BY:

   * Cost by number of children
   * Cost by gender
   * Cost by region
10. Compare cost differences between smokers and non-smokers across regions


## 📊 Key Insights

* Smokers tend to have higher insurance costs compared to non-smokers
* Higher BMI is associated with increased insurance costs
* Regional differences impact insurance pricing
* Age plays a significant role in determining premiums

## 🚀 Skills Demonstrated
